# BrandCraft

A sophisticated, modern hero section for a brand and UI/UX design portfolio featuring a striking split-layout with grayscale imagery, bold typography, and vibrant call-to-action elements. Designed to showcase design expertise while creating immediate visual impact.

## Live Demo

[View Live Demo](https://thisislefa.github.io/brandcraft)

## Features

- **Split-Layout Design**: Balanced image and text composition with optimal visual hierarchy
- **Grayscale Imagery**: Professional black and white photo treatment for sophisticated aesthetic
- **Bold Typography**: Large, impactful headlines with precise Inter font weights
- **Vibrant CTA**: Eye-catching orange action button with hover effects and icon integration
- **Responsive Design**: Seamlessly adapts from desktop to mobile with intelligent stacking
- **Performance Optimized**: Efficient CSS with custom properties and minimal dependencies
- **Accessibility**: Proper semantic structure, ARIA labels, and color contrast ratios
- **Modern CSS**: CSS Grid and Flexbox with aspect ratio controls and smooth transitions

## Tech Stack

- **HTML5** - Semantic structure with clean, accessible markup
- **CSS3** - CSS Grid, Flexbox, custom properties, and modern layout techniques
- **Google Fonts** - Inter font family (400-900 weights)
- **SVG Icons** - Custom arrow icon for button enhancement
- **Unsplash** - High-quality professional photography

## Usage

### Basic Integration
The hero section is entirely self-contained in a single HTML file. Simply include it in your project:

```html
<!-- Copy the entire hero-section-container -->
<div class="hero-section-container">
    <!-- Hero content -->
</div>
```

### Customizing Content
Update the hero content by modifying the text elements:

```html
<h1 class="headline-main">
    Your compelling headline here
</h1>

<p class="description-lead">
    Your professional introduction and value proposition.
</p>
```

## Customization

### Colors
- Dark Background: `#000000`, `#0d0d0d`
- Text Primary: `#ffffff`
- Text Secondary: `#c2c2c2`
- CTA Orange: `#ff5d17`
- CTA Hover: `#f76323`

### Typography
- Main Headline: 3.5rem (56px), 700 weight, -0.03em letter spacing
- Description Lead: 1.3rem (21px), 600 weight
- Button Text: 1rem (16px), 600 weight

### Layout
- Container Max Width: 1300px
- Image Ratio: 1 / 1.1 (reduced from original)
- Border Radius: 4px
- Content Padding: 3rem (text area)
- Max Height: 600px
